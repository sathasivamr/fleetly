# Fleetly

React + Vite SPA for Traccar fleet management: live tracking, reports, logistics (with optional demo data or your own API), and more.

## Quick start

```bash
cp .env.example .env
# Set VITE_TRACCAR_URL to your Traccar server (see .env.example)
npm install
npm run dev
```
